# kanboard-helm
Helm Chart to simplify the deployment of [Kanboard](https://github.com/kanboard/kanboard).

## Installation

```sh
helm repo add kanboard https://kube-the-home.github.io/kanboard-helm/
helm install kanboard/kanboard
```
